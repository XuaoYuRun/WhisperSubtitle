# WhisperProjectCN.py — 中文版

> **版本**：中文版（Chinese）  
> **文件路径**：`src/core/WhisperProjectCN.py`  
> **用途**：专门针对中文语音的转录优化，中文标点转换  
> **对应 GUI**：中文转录（`中文转录` 模式）

---

## 使用场景

当你需要处理以下类型的中文视频时，选择此版本：

- **中文语音内容**（普通话、方言等中文音频）
- **需要中文标点**（，。！？等）的转录输出
- **中文课程、讲座、播客、采访、会议** 等内容
- **保持原文语气和表达方式** 的需求

**不适用场景**：
- 英文内容（请使用英文标准版或防幻觉版）
- 需要英文标点的输出（英文版使用 `.` `!` `?`，中文版使用 `。` `！` `？`）

## 版本说明书

### 核心设计目标

中文版的核心目标是**准确识别中文语音并输出符合中文标点规范的文本**。它与英文版的主要区别：

1. **语言检测**：强制 `zh`（中文），而非 `en`（英文）
2. **标点转换**：自动将英文标点转换为中文标点（`.` → `。`，`!` → `！` 等）
3. **句子结束检测**：以 `。` `！` `？` 而非 `.` `!` `?` 作为句子结束标志
4. **初始提示**：使用中文提示，指导模型使用中文标点

### 后处理流程

```
视频输入 → Faster-Whisper 转录 → 句子合并（以中文标点为准） → 中文标点转换 → 输出 TXT
```

1. **转录**：使用 `large-v3-turbo` 模型，强制中文语言检测
2. **句子合并**：按 `。` `！` `？` 标点合并片段为完整句子
3. **中文标点转换**：
   - `,` → `，`（逗号）
   - `.` → `。`（句号）
   - `!` → `！`（感叹号）
   - `?` → `？`（问号）
   - `:` → `：`（冒号）
   - `;` → `；`（分号）
   - `"` → `「` / `」`（引号）
   - `(` → `（` / `）`（括号）
4. **输出**：每行一句，纯文本 TXT，无时间戳、无文件头

### 与英文版的区别

| 功能 | 英文版 | 中文版 |
|------|--------|--------|
| 语言检测 | `language="en"` | `language="zh"` |
| 初始提示 | 英文提示 | **中文提示** |
| 句子结束标志 | `.` `!` `?` | `。` `！` `？` |
| 后处理函数 | `ensure_proper_case`（首字母大写） | `ensure_chinese_punctuation`（中文标点转换） |
| 大小写修正 | 有（英文专有名词） | **无**（中文无大小写） |

### 输出位置

- 默认：视频所在目录的 `Text` 子文件夹
- 用户指定目录：使用 `-o` 参数指定
- 桌面保存（可选）：勾选"自动保存到桌面并转 Markdown"后，额外保存到 `C:\Users\Administrator\Desktop\Whisper语音列表\Text`

## 核心参数

### 转录参数（AI 模型推理层面）

| 参数 | 数值 | 作用说明 |
|------|------|----------|
| `language` | `"zh"` | 强制识别中文 |
| `task` | `"transcribe"` | 仅转录，不翻译 |
| `beam_size` | `5` | 束搜索宽度，5 条路径并行解码 |
| `best_of` | `5` | 从 5 个候选中选最优 |
| `patience` | `1.5` | 束搜索耐心值 |
| `length_penalty` | `1.0` | 长度惩罚系数 |
| `temperature` | `0.0` | 采样温度为 0，确定性输出 |
| `compression_ratio_threshold` | `2.4` | 压缩比阈值，检测重复文本 |
| `log_prob_threshold` | `-1.0` | 对数概率阈值，过滤低置信度 |
| `no_speech_threshold` | `0.6` | 无语音阈值 |
| `condition_on_previous_text` | `True` | 启用上下文依赖，提升连贯性 |
| `word_timestamps` | `False` | 不输出词级时间戳 |
| `vad_filter` | `True` | 启用语音活动检测 |
| `min_silence_duration_ms` | `300` | 300ms 静音视为断句点 |

### 与英文标准版参数对比

| 参数 | 英文标准版 | 中文版 | 差异说明 |
|------|------------|--------|----------|
| `language` | `"en"` | `"zh"` | 唯一不同：语言强制 |
| `compression_ratio_threshold` | `2.4` | `2.4` | 相同 |
| `log_prob_threshold` | `-1.0` | `-1.0` | 相同 |
| `no_speech_threshold` | `0.6` | `0.6` | 相同 |
| `condition_on_previous_text` | `True` | `True` | 相同 |
| `min_silence_duration_ms` | `300` | `300` | 相同 |

**结论**：中文版的 AI 模型参数与英文标准版**完全一致**，唯一的区别是 `language="zh"` 和中文提示。这确保了中文转录在连贯性和准确性上与英文标准版保持一致。

### 初始提示（Initial Prompt）

```
"请准确转录以下中文音频。"
"使用正确的中文标点符号。"
"确保每个句子完整，并以句号、感叹号或问号结尾。"
"保持原文的语气和表达方式。"
```

这段提示告诉模型：
- 准确转录中文音频
- 使用中文标点符号
- 确保句子完整，以 `。` `！` `？` 结尾
- 保持原文的语气和表达方式（不随意增删内容）

## 使用命令

```bash
# 单文件（自动创建 Text 目录）
python src/core/WhisperProjectCN.py "D:\视频\中文课程.mp4"

# 文件夹（递归扫描）
python src/core/WhisperProjectCN.py "D:\视频\中文课程文件夹"

# 强制指定输出目录
python src/core/WhisperProjectCN.py "D:\视频\中文课程.mp4" -o "D:\字幕"

# 同时保存到桌面
python src/core/WhisperProjectCN.py "D:\视频\中文课程.mp4" --desktop
```
